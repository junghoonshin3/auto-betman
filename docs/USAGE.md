# Auto Betman 사용법

## 시작하기

봇이 실행되면 설정한 채널에 **"Betman Tracker 시작됨 ✔"** 메시지가 표시됩니다.
채팅창에 `/`를 입력하면 사용 가능한 명령어 목록이 자동완성으로 나타납니다.

---

## 계정 등록 (`/setup`)

베트맨 구매내역을 조회하려면 먼저 계정을 등록해야 합니다.

### `/setup register` — 계정 등록

1. `/setup register` 입력
2. 모달 팝업에 아이디/비밀번호/알림방식 입력
3. 제출하면 등록 완료

> 알림 방식: `dm` (기본, 개인 메시지) 또는 `channel` (채널 멘션). DM 실패 시 자동으로 채널로 전환됩니다.

### `/setup status` — 등록 상태 확인

등록된 아이디, 알림 방식, 등록일을 확인합니다.

### `/setup remove` — 계정 해제

등록된 계정을 삭제합니다.

## 구매내역 조회 (`/purchases`)

등록된 베트맨 계정의 구매내역을 조회합니다. 모든 구매 건이 **하나의 메시지에 요약**되어 표시됩니다.

### 필터 옵션

| 옵션 | 설명 |
|------|------|
| `현재 회차 구매경기 (기본)` | 가장 최신 회차의 구매내역만 표시 |
| `전체` | 모든 구매내역 표시 |
| `최근 1시간` | 최근 1시간 이내 구매 건만 표시 |

### 메시지 형식

```
19회차 구매경기 4건

🎫 9B0A-CAA2-6461-DDF0
🟢 26.02.12(수) 10:04 | 5,000원 | 배당 4.20 | 예상 21,000원
`전북현대` vs `울산현대` → 승 (2.10)
`수원FC` vs `인천유나이티드` → 패 (2.00)

🎫 8B22-C941-2411-DB90
🟠 26.02.12(수) 10:30 | 3,000원 | 배당 3.50 | 예상 10,500원
`리버풀` vs `맨시티` → 무 (3.40)
`바르셀로나` vs `레알마드리드` → 승 (1.85)

총 4건 | 총 구매: 18,000원 | 총 예상적중: 63,000원
```

상태 아이콘: 🟢 발매중 | 🟠 발매마감 | 🏆 적중 | ❌ 미적중 | 🚫 취소

## 경기 목록 조회 (`/games`)

현재 프로토 승부식에서 구매 가능한 경기 목록을 조회합니다.

```
/games                # 전체 경기 목록
/games sport:축구     # 축구만
/games sport:농구     # 농구만
/games sport:야구     # 야구만
/games sport:배구     # 배구만
```

각 경기에 표시되는 정보: 경기번호, 리그, 게임유형(일반/핸디캡/언더오버), 홈vs원정, 배당, 마감시간, 경기장

---

## 통계 조회 (`/stats`)

| 옵션 | 설명 |
|------|------|
| `전체 (기본)` | 총 베팅수, 적중/미적중, 적중률, 총 구매/적중금액, 손익 |
| `일별 (7일)` | 최근 7일간 일별 베팅수, 적중수, 투자금액, 손익 |
| `월별 (6개월)` | 최근 6개월간 월별 현황 |

---

## 자동 알림

봇은 설정된 주기(기본 30분)마다 자동으로 스크래핑하여 알림을 보냅니다.

| 알림 | 시점 | 내용 |
|------|------|------|
| 구매 알림 | 새 구매 건 발견 | 경기 정보, 배당, 금액 |
| 결과 알림 | 경기 결과 확정 | 적중/미적중, 수익/손실 |

---

## 명령어 요약

| 명령어 | 설명 |
|--------|------|
| `/setup register` | 베트맨 계정 등록 |
| `/setup status` | 등록 상태 확인 |
| `/setup remove` | 계정 해제 |
| `/purchases` | 구매내역 조회 (현재 회차) |
| `/purchases filter:전체` | 전체 구매내역 조회 |
| `/purchases filter:최근 1시간` | 최근 1시간 구매내역 |
| `/games` | 구매 가능한 경기 목록 |
| `/games sport:축구` | 종목별 필터링 |
| `/stats` | 전체 통계 |
| `/stats period:일별 (7일)` | 일별 통계 |
| `/stats period:월별 (6개월)` | 월별 통계 |
